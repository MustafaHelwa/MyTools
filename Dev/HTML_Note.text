# This will be my notebook for HTML learning 
(from https://www.freecodecamp.org and other sources)

- Headline:     <h1>text</h1>
- Paragraph:    <p>text</p>
- Comment:      <!--comment-->
- main content: <main>cont</main>
- adding img:   <img src="URL" alt="description">
- link to another page: <a> href="URL">"Text" </a>
	- Link in text example: <p>See more <a href="https://freecatphotoapp.com">cat photos</a> in our gallery.</p>
- Open links in a new tab: target="_blank"
	- Example: <p>See more <a href="https://freecatphotoapp.com" target="_blank">cat photos</a> in our gallery.</p>
- Image into a link:
		<a href="example-link">
			<img src="image-link.jpg" alt="A photo of a cat.">
		</a>
- Sections: 	<section></section>
- Unordered list: <ul></ul>
- Elements of list: <li>Item</li>
- Self-contained content: <figure> </figure>
- Figure caption: <figcaption>Cap</figcaption>
- Emphasis a specific word: <em>word</em>
- Example: 
        <figure>
          <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg" alt="A slice of lasagna on a plate.">
          <figcaption>Cats <em>love</em> lasagna.</figcaption>  
        </figure>
- Ordered list: <ol></ol>
-Example:
        <ol>
          <li>flea treatment</li>
          <li>thunder </li>
          <li>other cats</li>
        </ol>
- Emphasis a Strong word: <strong>word</strong>
- get information from a user: <form action="/submit-url"></form>
- void element: <input>
- input type(field): 	<input type="value">
- Note that value may be "text" or other
- input name(field): 	<input type="text" name="name">
- input name example: 
      <section>
        <h2>Cat Form</h2>
        <form action="https://freecatphotoapp.com/submit-cat-photo">
          <input type="text" name="catphotourl" placeholder="cat photo URL">
        </form>
      </section>
- Not missable: required 
- Example: 
	<input required type="text" name="catphotourl" placeholder="cat photo URL">
- Button element: <button>Submit</button>
- Note that both input and button elements are inline elements, which don't appear on new lines.
- Other way of doing button:	<button type="submit">Submit</button>
- Radio buttons for specific answers: <input type="radio"> Indoor
- Label element:	<label> <input type="radio"> Indoor</label>
- id attribute: <input id="email">
- Note: To make one radio button automatically deselects the other both buttons must have a "name" attribute with the same value. 
- Example: 
          <label><input id="indoor" type="radio" name="indoor-outdoor"> Indoor</label>
          <label><input id="outdoor" type="radio" name="indoor-outdoor"> Outdoor</label>\
- value attribute example:
          <label><input id="indoor" value="indoor" type="radio" name="indoor-outdoor"> Indoor</label>
          <label><input id="outdoor" value="outdoor" type="radio" name="indoor-outdoor"> Outdoor</label>
- group related inputs and labels together: 
          <fieldset>
          <label><input id="indoor" type="radio" name="indoor-outdoor" value="indoor"> Indoor</label>
          <label><input id="outdoor" type="radio" name="indoor-outdoor" value="outdoor"> Outdoor</label>
          </fieldset>
- Adding legend to the list: 
          <fieldset>
            <legend>Is your cat an indoor or outdoor cat?</legend>
            <label><input id="indoor" type="radio" name="indoor-outdoor" value="indoor"> Indoor</label>
            <label><input id="outdoor" type="radio" name="indoor-outdoor" value="outdoor"> Outdoor</label>
          </fieldset>
- To create a checkbox: 
            <legend>What's your cat's personality?</legend>
            <input type="checkbox"> Loving
- Another way to use for instead of id element: 
	   <input id="loving" type="checkbox"> <label for="loving">Loving</label>
- Adding name to the input:
            <input id="loving" type="checkbox" name="personality"> <label for="loving">Loving</label>            
            <input id="lazy" type="checkbox" name="personality"> <label for="lazy">Lazy</label>
