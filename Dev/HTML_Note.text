# This will be my notebook for HTML learning 
(from https://www.freecodecamp.org and other sources)

- Headline:     <h1>text</h1>
- Paragraph:    <p>text</p>
- Comment:      <!--comment-->
- main content: <main>cont</main>
- adding img:   <img src="URL" alt="description">
- link to another page: <a> href="URL">"Text" </a>
	- Link in text example: <p>See more <a href="https://freecatphotoapp.com">cat photos</a> in our gallery.</p>
- Open links in a new tab: target="_blank"
	- Example: <p>See more <a href="https://freecatphotoapp.com" target="_blank">cat photos</a> in our gallery.</p>
- Image into a link:
		<a href="example-link">
			<img src="image-link.jpg" alt="A photo of a cat.">
		</a>
- Sections: 	<section></section>
- Unordered list: <ul></ul>
- Elements of list: <li>Item</li>
- Self-contained content: <figure> </figure>
- Figure caption: <figcaption>Cap</figcaption>
- Emphasis a specific word: <em>word</em>
- Ordered list: <ol></ol>
